## Описание проекта
Проект посвящен анализу данных фермерского хозяйства и разработке моделей для прогнозирования удоя коров и вкуса их молока. Это позволит фермеру выбрать коров с наибольшим потенциалом по удою и вкусовым качествам молока, минимизируя риски при покупке.

## Цель проекта
Спрогнозировать годовой удой и вероятность получения вкусного молока, чтобы отобрать коров с удоем не менее 6000 кг и высокими вкусовыми качествами молока.

## Результаты
* Построены и сравнены три модели линейной регрессии для предсказания удоя.
* Выявлены ключевые факторы, влияющие на удой: ЭКЕ, СПО, возраст коровы и порода папы.
* Обучена модель логистической регрессии для классификации вкуса молока.
* Получены прогнозы удоя и вкуса молока для коров, планируемых к покупке.

## Рекомендации
* При покупке коров учитывать прогнозируемый удой и вероятность вкусного молока.
* Добавить данные о здоровье коров, условиях содержания и дополнительные параметры молока для повышения точности прогнозов.
* Для увеличения удоя рекомендуется повысить питательность корма (ЭКЕ и СПО).
