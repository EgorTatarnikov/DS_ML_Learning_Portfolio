# Проект: [Выбор региона для нефтеразработки]()
## Описание проекта
Проект посвящён выбору оптимального региона для разработки новых нефтяных скважин. Использовались данные геологоразведки и методы машинного обучения для прогнозирования запасов нефти и оценки потенциальной прибыли с учётом рисков.

## Цель проекта
Построить модель для прогнозирования запасов нефти в скважинах трёх регионов и определить регион с максимальной прибылью и минимальными рисками.

## Результаты
* Проанализированы данные о запасах нефти в трёх регионах.
* Построена модель и оценены запасы нефти.
* Произведен расчёт прибыли и оценка рисков.
* Определён наиболее выгодный регион, с высокой прибылью и низким риском убытков.

## Рекомендации
Рекомендуется развивать добычу в регионе 1, так как он обеспечивает наибольшую прибыль при минимальных рисках.

## Приобретенные навыки
* EDA
* Расчёт прибыли и оценка рисков с использованием Bootstrap
* Интерпретация результатов машинного обучения для принятия бизнес-решений.
